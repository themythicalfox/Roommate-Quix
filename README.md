# Roommate Protocol

30-question roommate compatibility instrument. Six archetypes, weighted compatibility
engine, optional profile intake (age, gender, MBTI, enneagram, faith).
Next.js 16 + React 19. No database. No env vars.

## Run locally
```
npm install
npm run dev
```
Open http://localhost:3000

## Deploy to Vercel
1. Push this folder to a new GitHub repo.
2. vercel.com → Add New → Project → import the repo → Deploy.

Or with the Vercel CLI:
```
npm i -g vercel
vercel
```
